# Skin Presets

A Geometry Dash mod built with the **Geode SDK** that allows you to save, manage, and load your favorite icon and color combinations directly from the garage menu!

## Features

* **Save Custom Presets:** Instantly save your current cube, ship, ball, UFO, wave, robot, spider, swing, colors, glow, and death effect under a custom name.
* **Unlimited Storage:** Thanks to a clean, integrated native scrolling list, you can save as many presets as you want.
* **Visual Preview:** See a miniature preview of your custom cube icon right next to the preset name before loading it.
* **Seamless Integration:** The mod adds a neat "+" button directly inside the official `GJGarageLayer` (the skin selection menu) for a native feel.

## Preview

The mod adds an interface with a custom input box to name your creation, a green "Save" button, and a scrollable list with "Load" and "Delete" actions for each profile.

## How to Install

1. Open Geometry Dash with **Geode** active.
2. Go to the Geode Mod Index.
3. Search for **Skin Presets** and click **Install**.
4. Restart your game and open the Icon Kit / Garage menu to start saving your styles!


## How to Use

1. Open Geometry Dash with **Geode** active.
2. Go to the garage Menu.
3. Search for the **+** icon next to the **swingcopter** button.
4. You need to enter a name for your preset either you wont be able to save your preset !$
